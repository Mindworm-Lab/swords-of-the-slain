# Swords of the Slain

A rogue-like MMO RPG featuring animated line-of-sight fog-of-war, procedural dungeons, and a visually compelling navigable gamespace.

## Tech Stack

- **Frontend**: Vite + React + TypeScript + PixiJS v8
- **Backend**: Rust + Axum
- **Deployment**: Docker + NGINX

## Development

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
cargo run
```

## Deployment

```bash
docker compose up --build
```
